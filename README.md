## 快速安裝
### 前置條件
#### 伺服器最小配寘：
CPU: 2核  
記憶體：4G  
存儲：40G
#### 作業系統：
CentOS  
Ubuntu  
### 安裝部署
#### 一、下載安裝程式
```
~# curl -OL https://github.com/bluewhalekefu/kefu/releases/download/1.0/kefu_1.0.tar.gz
```
#### 二、安裝
1.解壓
```
~# tar -zxvf kefu_ 1.0.tar.gz
```
2.進入安裝程式目錄
```
~# cd kefu_ 1.0
```
3.執行inst_ all.sh
```
~# sh inst_ all.sh
```
#### 三、訪問
```
http://[ip/domainName]/admin
用戶名：admin 密碼：123456
```
